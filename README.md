# 🗿 MOAI SAVES — DeFi Rescue

A fast-paced browser game where you defend the DeFi vault from liquidators!

## Deploy to Vercel

### Option 1: Vercel CLI (Fastest)
```bash
npm i -g vercel
vercel
```

### Option 2: Vercel Dashboard (No CLI needed)
1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Drag & drop this folder onto the dashboard
3. Click **Deploy** — done!

### Option 3: GitHub Integration
1. Push this folder to a GitHub repo
2. Import the repo on Vercel
3. Auto-deploys on every push ✅

## Game Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move | ← → Arrow Keys | Swipe or ◀▶ buttons |
| Smash | Z or Space | 💥 button |
| Shield | X | 🛡️ button |

## Notes
- Leaderboard scores are saved in **browser localStorage** (per-device)
- No backend required — fully static deployment
- Works on desktop and mobile
